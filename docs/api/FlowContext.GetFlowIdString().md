## FlowContext\.GetFlowIdString\(\) Method

Gets the flow ID as a string in format "N" \(32 digits without hyphens\)\.
Useful for logging and correlation\.

```csharp
public string GetFlowIdString();
```

#### Returns
[System\.String](https://learn.microsoft.com/en-us/dotnet/api/system.string 'System\.String')  
The flow ID as a 32\-character hexadecimal string\.
